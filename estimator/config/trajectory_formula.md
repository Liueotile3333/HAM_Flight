# 期望轨迹分段公式（sim_odom / add_traj_bias.cpp）

三段式期望轨迹：**垂直爬升 → 圆/8字机动 → 原地垂直降落**。
时间 `t` 从触发时刻（`pub_triger.sh` 发 `/landing_triger`）起算，即代码中的 `time_diff`。

## 参数符号

| 符号 | yaml 参数 (odom_param.yaml) | 默认值 | 含义 |
|------|---------------------------|--------|------|
| `T_r` | `traj_rise_time` | 5 s | 爬升时长 |
| `T_m` | `traj_duration` | 50 s | 机动段时长 |
| `T_d` | `traj_descend_time` | 5 s | 降落时长 |
| `h` | `traj_height` | 2 m | 爬升高度（相对起飞点） |
| `R` | `traj_radius` | 5 m | 圆半径 / 8字 x半幅 |
| `A` | `traj_amp_y` | 5 m | 8字 y半幅（圆忽略） |
| `ω` | `traj_omega` | 0.2 rad/s | 角速度（圆线速度 = R·ω） |
| `type` | `traj_type` | 1 | 1=圆, 2=8字 |
| `(x₀,y₀,z₀)` | — | 起飞点 | 触发时锁定的 `current_position` |

总时长 = `T_r + T_m + T_d`（默认 60 s）。

## 平滑函数（minimum-jerk，保证位置/速度/加速度全程连续）

```
s(r)   = 10r³ − 15r⁴ + 6r⁵          ∈ [0,1],  r ∈ [0,1]
s'(r)  = 30r²(1−r)²                   (一阶导)
s''(r) = 60r(1−r)(1−2r)               (二阶导)
```

性质：`s(0)=0, s(1)=1, s'(0)=s'(1)=0, s''(0)=s''(1)=0`
→ 段间**位置/速度/加速度全连续**（余弦平滑 s''(端点)≠0 会有加速度跳变，故改用 minimum-jerk）。

## 时间分点

```
t₁ = T_r
t₂ = T_r + T_m
t₃ = T_r + T_m + T_d
```

---

## 段 1：垂直爬升　(0 ≤ t < t₁)

```
x(t) = x₀
y(t) = y₀
z(t) = z₀ + h·s(t/T_r)

ẋ = 0,  ẏ = 0
ż = (h / T_r)·s'(t/T_r)

ẍ = 0,  ÿ = 0
z̈ = (h / T_r²)·s''(t/T_r)
```
xy 保持起飞点，仅 z 平滑爬升到 `z₀ + h`。

---

## 段 2：机动（圆 / 8字）　(t₁ ≤ t < t₂)

令 `τ = t − t₁`，`r_m = τ/T_m`，**平滑相位** `φ = ω·T_m·s(r_m)`（使起止速度/加速度=0，与垂直段连续）。
记 `φ' = ω·s'(r_m)`，`φ'' = (ω/T_m)·s''(r_m)`。高度恒定 `z = z₀ + h`。

### 圆（traj_type = 1）
> x 用 `(cos φ − 1)` 使 φ=0 时 x=x₀（与爬升终点衔接）；φ 用平滑相位使起止速度=0（与垂直段连续）。
```
x = x₀ + R·(cos φ − 1),    y = y₀ + R·sin φ
ẋ = −R·sin φ · φ',         ẏ =  R·cos φ · φ',              ż = 0
ẍ = −R·cos φ·φ'² − R·sin φ·φ'',   ÿ = −R·sin φ·φ'² + R·cos φ·φ'',   z̈ = 0
```

### 8字（traj_type = 2，Lissajous）
```
x = x₀ + R·sin φ,         y = y₀ + A·sin(2φ)
ẋ =  R·cos φ · φ',        ẏ = 2A·cos(2φ) · φ',           ż = 0
ẍ = −R·sin φ·φ'² + R·cos φ·φ'',  ÿ = −4A·sin(2φ)·φ'² + 2A·cos(2φ)·φ'',  z̈ = 0
```

**起止（r_m=0 或 1）**：`s'=s''=0 → φ'=φ''=0 → 速度=0、加速度=0`，与爬升/降落段无缝衔接。
**速度幅值**：平均线速度 = R·ω；峰值 = R·ω·1.875（s' 峰值 1.875）。

---

## 段 3：原地垂直降落　(t₂ ≤ t < t₃)

机动结束位置 `(x_L, y_L)`：用 `θ_end = ω·T_m` 代入段 2 的位置公式得到，降落段保持不变。

```
x(t) = x_L,   y(t) = y_L
令 r_d = (t − t₂) / T_d
z(t) = (z₀ + h) − h·s(r_d)

ẋ = 0, ẏ = 0
ż = −(h / T_d)·s'(r_d)

ẍ = 0, ÿ = 0
z̈ = −(h / T_d²)·s''(r_d)
```
xy 锁定机动结束位置，仅 z 平滑降回 `z₀`。

---

## 段 4：结束　(t ≥ t₃)

```
x = x_L,  y = y_L,  z = z₀
速度 = 0,  加速度 = 0
```

---

## future（前瞻 0.5 s，供 controller 算 yaw）

用 `t_f = t + 0.5` 代入上述同分段公式，仅取 position / velocity（不需要加速度）。
controller 的 `calculate_yaw_velo` 用 `future_velocity` 的 `atan2(vy, vx)` 算期望偏航：
- 爬升/降落段速度 xy=0 → yaw 保持上一帧
- 机动段 → yaw 自动跟随圆/8字切线方向

---

## 备注

- **jerk**：各段均设为 0（代码 `final_jerk.setZero()`），未建模高阶项。
- **段间连续性**：由 `s'(0)=s'(1)=0` 保证，段 1↔2、段 2↔3 在交界处速度连续（xy 速度从 0 平滑过渡到机动速度、再回到 0）。
- **降落点** `(x_L, y_L)` 由 `ω·T_m`（机动结束相位）决定：
  - 想降回起飞点附近 → 取整数圈：`ω·T_m = 2π·n`（如 ω=0.2 → T_m ≈ 31.4s 为 1 圈）。
  - 仿真可任意；真机注意降落点不要撞障碍物。
- **vel_scale**：若 `vel_scale≠1.0`，最终 cmd 的位移/速度/加速度会再整体 ×vel_scale（轨迹范围与速度同比例缩放，时间不变）。
